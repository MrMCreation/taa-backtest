# Tactical Asset Allocation Backtest

A full-featured Tactical Asset Allocation (TAA) backtest web application that replicates Portfolio Visualizer's TAA models, powered by Yahoo Finance data.

## Live Demo

**[Open TAA Backtest](https://mrmcreation.github.io/taa-backtest/)**

## Features

- 7 TAA Models: Moving Average, Dual Momentum, Relative Strength, Adaptive Allocation, Target Volatility, Market Valuation, Seasonal
- - Performance Metrics: CAGR, Sharpe, Sortino, Max Drawdown
  - - Charts: Equity curve, drawdowns, annual returns, monthly heatmap, rolling returns, allocation
   
    - ## Data Source
   
    - Yahoo Finance (real-time via browser)
